# RailsIntroduce
Project codes while I'm learning ruby on rails and ruby programming language.

# Connect to rails console

```bash
>> rails console
```

# Produce web server on local

```bash
>> rails server
```

# Migrate database

* Drop the database

  ```bash
  >> rails db:drop
  ```

* Recreate the database

  ```bash
  >> rails db:create
  ```

* Migrate database

  ```bash
  >> rails db:migrate
  ```
